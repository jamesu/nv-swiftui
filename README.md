# nv-swiftui

This is a 99.9% vibe coded swiftui rewrite of the notational velocity codebase as taken from `scrod/nv`.

To run currently, execute the following in terminal:

	swift run NVRewrite

Right now core functionality is implemented, however there are likely a lot of minor things and missing features that have been overlooked, so stay tuned!
